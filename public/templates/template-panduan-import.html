Panduan Import Data Pengguna

Untuk mengimpor data pengguna, silakan gunakan template CSV yang tersedia. Berikut adalah panduan untuk mengisi template:

Kolom Keterangan Contoh
Nama* Nama lengkap pengguna John Doe
Email* Alamat email (harus unik) john.doe@example.com
Username Username untuk login (opsional, harus unik jika diisi) johndoe
Password* Password untuk login rahasia123
Peran* Peran pengguna (harus sesuai dengan peran yang tersedia di sistem) Karyawan
Nomor Telepon Nomor telepon (opsional) 08123456789
Notifikasi WA (Ya/Tidak) Apakah pengguna ingin menerima notifikasi via WhatsApp (opsional, default: Tidak) Ya

Catatan:

  • Kolom yang ditandai dengan * adalah wajib diisi.
  • Format file yang didukung adalah CSV, XLS, dan XLSX.
  • Pastikan nilai di kolom "Peran" sesuai dengan peran yang tersedia di sistem.
  • Kolom "Notifikasi WA" hanya menerima nilai "Ya" atau "Tidak". Nilai lain akan dianggap "Tidak".
  • Ukuran file maksimal adalah 2MB.

Panduan XLSX/XLS:

  1. Download template CSV yang disediakan.
  2. Buka dengan Microsoft Excel, Google Sheets, atau aplikasi spreadsheet lainnya.
  3. Isi data sesuai dengan format.
  4. Simpan sebagai format XLSX atau XLS.
  5. Upload file yang telah disimpan.

Contoh isi file CSV:

Nama,Email,Username,Password,Peran,Nomor Telepon,Notifikasi WA (Ya/Tidak)
John Doe,john.doe@example.com,johndoe,rahasia123,Karyawan,08123456789,Ya
Jane Smith,jane.smith@example.com,janesmith,rahasia456,Manager,08198765432,Tidak
